## Cara Menjalankan Aplikasi

Untuk menjalankan aplikasi MoodSync di komputer Anda, pertama lakukan clone repository dari GitHub dengan menjalankan perintah berikut:

### 1. Clone repository dari GitHub dengan perintah
```bash
git clone https://github.com/syafwan-ux/MoodSync.git
```

### 2. Masuk ke direktori project
```bash
cd MoodSync
```

### 3. Pindah ke branch yang sesuai
```bash
git checkout version0.2
```

### 4. Masuk ke folder aplikasi
```bash
cd MoodSync_version0.1
```

### 5. Jalankan aplikasi dengan Python
```bash
python main.py
```

# Catatan

- Aplikasi ini direkomendasikan menggunakan Python versi 3.10.x.
- Pastikan Python sudah terinstall dan dapat diakses melalui terminal (Command Prompt).
- Jika terjadi error karena modul tidak ditemukan, jalankan perintah berikut:
  pip install -r requirements.txt
- Pada beberapa sistem, gunakan "python3 main.py" jika perintah "python" tidak dikenali.


Selesai. Aplikasi seharusnya sudah dapat dijalankan dengan baik.
