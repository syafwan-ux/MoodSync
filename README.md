# MoodSync Application

## Cara Menjalankan Aplikasi

Untuk menjalankan aplikasi MoodSync di komputer Anda, pertama lakukan clone repository dari GitHub dengan menjalankan perintah berikut:

### 1. Clone repository dari GitHub dengan perintah
```bash
git clone https://github.com/syafwan-ux/MoodSync.git
```

### 2. Masuk ke direktori project
```bash
cd MoodSync
```

### 3. Pindah ke branch yang sesuai
```bash
git checkout version0.2
```

### 4. Masuk ke folder aplikasi
```bash
cd MoodSync_version0.1
```

### 5. Jalankan aplikasi dengan Python
```bash
python main.py
```
